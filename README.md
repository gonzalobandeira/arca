## Getting Started

### Prerequisites

- Xcode 15.0 or later
- iOS 17.0+ (for the latest SwiftUI features)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/gonzalobandeira/arca.git
   ```
2. Open `Arca/Arca.xcodeproj` in Xcode.
3. Select your target device/simulator and press `Cmd + R` to run.

## Technical Details

- **Language**: Swift
- **Framework**: SwiftUI
- **Security**: Local Authentication (Biometrics) & Keychain/Secure Storage.
- **OCR**: Native Vision framework for fast and accurate text extraction.
